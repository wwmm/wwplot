Features
  • XY and histogram plot
  • Linear and nonlinear fit
  • Import and export table in TSV format (tab separated values)
Required Libraries
  • Python 3
  • Numpy
  • Scipy
  • PyGobject 3
  • Cairocffi
  • Matplotlib
  • Gtk 3.18 or above
Program Execution
After installing the required libraries navigate to the program folder in a terminal and run this command:

python main.py
